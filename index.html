Algoritma Naive Bayes Achy-Nova

Halaman ini dibuat sebagai dokumentasi implementasi algoritma Naive Bayes dalam mengklasifikasikan tingkat kegemaran membaca masyarakat di setiap provinsi di Indonesia berdasarkan data tahun 2024.

Dataset sebelum SMOTE

Dataset dalam format Microsoft Excel merupakan data awal yang digunakan pada penelitian ini. Dataset tersebut berisi data tingkat kegemaran membaca masyarakat di setiap provinsi sebelum dilakukan proses preprocessing menggunakan metode SMOTE.

Lihat Dataset Excel

Notebook Google Colab

Notebook Google Colab digunakan sebagai tempat menjalankan seluruh proses penelitian. Di dalamnya terdapat proses membaca dataset, preprocessing menggunakan metode SMOTE, pembagian data training dan testing, pembentukan model Naive Bayes, hingga proses evaluasi menggunakan Accuracy, Precision, Recall, F1-Score, Classification Report, dan Confusion Matrix.

Buka Notebook Google Colab

Preprocessing Data (SMOTE)

Data yang telah diproses kemudian dibagi menjadi data training sebesar 80% dan data testing sebesar 20%. Data training digunakan untuk membangun model, sedangkan data testing digunakan untuk mengetahui kemampuan model dalam melakukan klasifikasi.

Hasil Evaluasi Moedel

Hasil pengujian menunjukkan bahwa model Naive Bayes mampu memberikan hasil klasifikasi yang baik. Berdasarkan proses evaluasi diperoleh nilai Accuracy sebesar 91,67%, Precision 88,89%, Recall 100%, dan F1-Score 94,12%. Selain itu, hasil klasifikasi juga ditampilkan dalam bentuk Classification Report dan Confusion Matrix untuk melihat performa model secara lebih rinci.\
